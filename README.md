# MiniMind 🌌 —— 深度解构与实战教程

> 🔍 **深入浅出，重构理解** —— 基于 [MiniMind](https://github.com/jingyaogong/minimind) 的大语言模型（LLM）架构拆解系列  
> 🌱 从 Tokenizer 到 MoE，从 Pretrain 到 Distillation，手把手带你构建属于自己的大模型

---

本项目是对开源项目 [jingyaogong/minimind](https://github.com/jingyaogong/minimind) 的深度学习笔记与源码实战。我们致力于在原有代码的基础上，进行**逐行级别的源码解析**，帮助开发者不仅能够“跑通”模型，更能从底层逻辑上“读懂”模型。

### ✨ 核心价值
- 🔬 **深度解构**：涵盖公式推导、实现逻辑以及训练工程中那些“秘而不宣”的隐藏细节。
- 🔁 **流程可视化**：为核心模块提供清晰的 `Shape` 转换注释，直观呈现每一层的数据流动。
- 🧠 **原理驱动**：拒绝黑盒，不仅教你“如何实现”，更探讨“为什么要这样设计”。
